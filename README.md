# Project_Planning_Stage_Individual-1
# Minecraft Player Newsletter Subscription Prediction - Planning Report

## Project Description
This project analyzes Minecraft server player data to predict newsletter subscription behavior based on player characteristics.

## Files in this Repository
- `report.ipynb` - Main Jupyter notebook with analysis
- `players.csv` - Player characteristics dataset
- `sessions.csv` - Play session data
- `README.md` - This file

## How to Run
1. Clone this repository
2. Make sure `players.csv` and `sessions.csv` are in the same directory as the notebook; and place into a folder called `data`
3. Open `report.ipynb` in Jupyter Notebook with R kernel
4. Run all cells in order from top to bottom
