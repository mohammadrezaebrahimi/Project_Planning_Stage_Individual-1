# Project_Planning_Stage_Individual-1
# Minecraft Player Newsletter Subscription Prediction - Planning Report

## Project Description
This project analyzes Minecraft server player data to predict newsletter subscription behavior based on player characteristics.

## Files in this Repository
- `INDIVIDUAL_STAGE_PROJECT.ipynb` - Main Jupyter notebook with analysis
- `players.csv` - Player characteristics dataset
- `sessions.csv` - Play session data
- `README.md` - This file

## Requirements
- R version 4.0 or higher
- Required R packages:
  - tidyverse
  - repr

## Installation Instructions

### Install R packages:
```r
install.packages("tidyverse")
install.packages("repr")
```

## How to Run
1. Clone this repository
2. Make sure `players.csv` and `sessions.csv` are in the same directory as the notebook
3. Open `INDIVIDUAL_STAGE_PROJECT.ipynb` in Jupyter Notebook with R kernel
4. Run all cells in order from top to bottom
